# ExternalFindingSeverity

ExternalFindingSeverity describes the severity of an external finding (Vulnerability or Security Alert)

## Example Usage

```java
import com.vanta.vanta_auditor_api.models.components.ExternalFindingSeverity;

ExternalFindingSeverity value = ExternalFindingSeverity.CRITICAL;
```


## Values

| Name       | Value      |
| ---------- | ---------- |
| `CRITICAL` | CRITICAL   |
| `HIGH`     | HIGH       |
| `LOW`      | LOW        |
| `MEDIUM`   | MEDIUM     |
