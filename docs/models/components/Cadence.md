# Cadence

How frequently this information request recurs (e.g., annual password
policy reviews). Null for one-time requests.

## Example Usage

```java
import com.vanta.vanta_auditor_api.models.components.Cadence;

Cadence value = Cadence.ANNUALLY;
```


## Values

| Name         | Value        |
| ------------ | ------------ |
| `ANNUALLY`   | ANNUALLY     |
| `BIANNUALLY` | BIANNUALLY   |
| `MONTHLY`    | MONTHLY      |
| `QUARTERLY`  | QUARTERLY    |
