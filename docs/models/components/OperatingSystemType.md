# OperatingSystemType

The possible types of the operating system. One of `mac_OS`, `linux`, or `windows`.

## Example Usage

```java
import com.vanta.vanta_auditor_api.models.components.OperatingSystemType;

OperatingSystemType value = OperatingSystemType.MAC_OS;
```


## Values

| Name      | Value     |
| --------- | --------- |
| `MAC_OS`  | macOS     |
| `LINUX`   | linux     |
| `WINDOWS` | windows   |
