# InformationRequestCadence

Frequency cadence for the information request, indicating how often it recurs.

## Example Usage

```java
import com.vanta.vanta_auditor_api.models.components.InformationRequestCadence;

InformationRequestCadence value = InformationRequestCadence.ANNUALLY;
```

| ------------ | ------------ |
| `ANNUALLY`   | ANNUALLY     |
| `BIANNUALLY` | BIANNUALLY   |
| `MONTHLY`    | MONTHLY      |
| `QUARTERLY`  | QUARTERLY    |
