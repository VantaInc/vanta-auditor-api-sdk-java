# IssueSnapshotItemOrderBy

Allowed values for the `orderBy` query parameter on issue snapshot item endpoints.

## Example Usage

```java
import com.vanta.vanta_auditor_api.models.components.IssueSnapshotItemOrderBy;

IssueSnapshotItemOrderBy value = IssueSnapshotItemOrderBy.CREATED_AT;
```


## Values

| Name               | Value              |
| ------------------ | ------------------ |
| `CREATED_AT`       | createdAt          |
| `LAST_MODIFIED_AT` | lastModifiedAt     |
